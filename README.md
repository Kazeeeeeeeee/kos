# KOS

Kos is an small personal operating system that I made because I am bored after the DSE examination

# Installation

It can't be actually installed so far, but if you wanna run it on your Linux QEMU for fun, it is pretty simple

```bash
./build.sh
```
And that's it, now you can enjoy one of the smallest OS in the world

# Command
Print text (e.g. echo hello world)
```bash
echo
```
Look for available command
```bash
help
```
Show OS version and information
```bash
info
```
Clear the terminal screen
```bash
clear
```
